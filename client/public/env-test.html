Environment Variables Test

This page tests if environment variables are being passed correctly during build.

Build Information:

Instructions:

  1. Check Vercel Dashboard → Settings → Environment Variables
  2. Ensure you have: VITE_CLERK_PUBLISHABLE_KEY (NOT NEXT_PUBLIC_...)
  3. The value should start with pk_live_ or pk_test_
  4. After adding/updating, redeploy by pushing any commit

Expected Variables in Vercel:

VITE_CLERK_PUBLISHABLE_KEY = [Your Clerk publishable key starting with pk_live_]
CLERK_SECRET_KEY = [Your Clerk secret key starting with sk_live_]  
DATABASE_URL = [Your PostgreSQL connection string]
